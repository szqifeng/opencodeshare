# LSP 服务器配置

OpenCode 通过 LSP（Language Server Protocol）服务器与主流编辑器深度集成，为您提供实时的代码补全、错误提示、重构建议等功能。本文将详细介绍 LSP 服务器的配置方法、支持编辑器列表、集成步骤以及常见问题解决。

通过配置 LSP 服务器，您可以在熟悉的编辑器中使用 OpenCode 的强大功能，无需切换工具，大幅提升开发效率。
