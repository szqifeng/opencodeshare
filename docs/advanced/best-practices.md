# 最佳实践

OpenCode 使用最佳实践指南。

## 配置优化

### 模型选择

根据任务类型选择合适的模型：
- 简单任务: claude-haiku
- 平衡任务: claude-sonnet-4-5
- 复杂推理: claude-opus-4-5-thinking

### 温度参数

| 场景 | 温度值 |
|------|--------|
| 代码生成 | 0.2-0.4 |
| 创意写作 | 0.7-0.9 |
| 问题分析 | 0.1-0.3 |

## 性能优化

- 减少上下文长度
- 使用局部索引
- 合理使用缓存
