# OpenCode Docs 需求文档

## 一、项目概述

- **项目名称**: OpenCode Docs
- **类型**: 教育 / 工具教程型官网
- **目标用户**: 开发者、技术爱好者
- **核心价值**: 易学 + 免费 + 实战

---

## 二、整体布局结构

### 1. 顶部导航栏（Header）

**结构**：
- 左：Logo + 网站名
- 中：主导航（开始学习 / 场景实战 / 进阶手册 / 社群）
- 右：搜索 + 语言切换 + 主题切换 + GitHub

**设计特点**：
- 极简横向布局
- 浅灰背景
- 高度约 64px
- 内容居中（最大宽度 1200px）
- 固定顶部（scroll sticky）

### 2. Hero 首屏区（核心卖点区）

**左右结构（50% / 50%）**：
- 左侧：大标题 + 副标题 + 3个按钮（主CTA + 次CTA）
- 右侧：插画（手绘风）

**风格特征**：
- 大留白
- 高对比标题
- 绿色主按钮（强调）
- 插画柔和色系

### 3. 核心优势区（三栏卡片）

三列等宽卡片，每个包含：
- 图标
- 标题
- 简短说明
- 链接按钮

---

## 三、视觉风格

### 颜色系统
- **主色**: 清爽绿色（偏薄荷绿）
- **强调色**: 蓝 / 粉
- **背景**: #f5f6f7 级别浅灰
- **文字**: 深灰而非纯黑

### 排版逻辑
- 无衬线字体
- 标题超大字重（Bold）
- 行距宽松
- 模块之间间距大（80px+）

### 插画风格
- 扁平手绘
- 低饱和度
- 情绪友好（亲和感）

---

## 四、交互模式

- 按钮 Hover 变深
- 卡片 Hover 微阴影
- 轻微过渡动画（200ms）
- 无复杂动效（克制）

---

## 五、技术建议

- **前端框架**: React / Vue
- **样式方案**: Tailwind 或 UnoCSS
- **支持**: 暗黑模式
- **插画**: SVG 格式

---
