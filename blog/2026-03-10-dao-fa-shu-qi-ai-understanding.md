可以用四个问题理解：

**为什么龙虾能帮我们做事 → 如何让龙虾做事 → 怎么让龙虾做好事情 → 有哪些好用的工具**

对应：

**道（原理） → 法（方法） → 术（技巧） → 器（工具）**

---

## 核心笔记

- **道（原理）**：AI 能思考、写代码、用工具的三种能力
- **法（方法）**：给 AI 清晰的任务流程和规则
- **术（技巧）**：用清晰指令和拆分任务与 AI 沟通
- **器（工具）**：AI 工具、技术环境、工具系统

## 学完你能做什么

- 理解 AI 为什么能帮你做事
- 学会如何让 AI 执行复杂任务
- 掌握与 AI 高效沟通的技巧
- 选择合适的工具组合提升效率

---

## 一、道（原理）：为什么龙虾能帮我们做事

![道（原理）- AI能思考、写代码、用工具的三种能力](https://opencodeshare.oss-cn-shenzhen.aliyuncs.com/ai-principle-three-abilities.png)

"道"是底层原理。

龙虾之所以能完成任务，是因为它有三种能力：

### 1. 会写代码

龙虾学过大量代码，可以：

- 写程序
- 修改程序
- 生成脚本

### 2. 会思考

龙虾可以：

- 分析问题
- 规划步骤
- 按步骤完成任务

例如：

```
读取数据 → 分析 → 生成报告
```

### 3. 会用工具

龙虾可以调用工具：

- 读写文件
- 上网查询
- 调用 API
- 运行代码

所以龙虾不只是聊天工具，而是：

**能思考、写代码、用工具的数字助手。**

这就是 **道（原理）**。

---

## 二、法（方法）：如何让龙虾做事

![法（方法）- 给AI清晰的任务流程和规则](https://opencodeshare.oss-cn-shenzhen.aliyuncs.com/ai-methods-task-flow-rules.png)

"法"是 **方法和做法**。

简单说就是：

**给龙虾设计一套做事的方法。**

例如：

分析 CSV：

1. 读取数据
2. 统计指标
3. 生成图表
4. 输出总结

这就是一套 **完成任务的方法**。

在 AI 里，本质就是：

**把任务拆成一个清晰的工作流程。**

这就是 **法（方法）**。

---

## 三、术（技巧）：怎么让龙虾做好事情

![术（技巧）- 用清晰指令和拆分任务与AI沟通](https://opencodeshare.oss-cn-shenzhen.aliyuncs.com/ai-techniques-clear-instructions.png)

"术"是 **使用技巧**。

例如：

### 写清晰的指令

**坏指令：**

```
分析数据
```

**好指令：**

```
读取 data.csv
统计平均值
生成图表
输出总结
```

或者：

**把复杂任务拆成步骤。**

同样一只龙虾，有的人用得很好，有的人用得很乱，差别往往就在这些 **使用技巧**。

这就是 **术（技巧）**。

---

## 四、器（工具）：有哪些好用的工具

![器（工具）- AI工具、技术环境、工具系统](https://opencodeshare.oss-cn-shenzhen.aliyuncs.com/ai-tools-technology-environment.png)

"器"就是 **工具本身**。

例如：

### AI 工具

- ChatGPT
- Claude
- OpenClaw

### 技术环境

- Python
- Node.js
- Docker

### 工具系统

- 文件系统
- API
- 搜索引擎
- 自动脚本

这些工具，就是龙虾干活用的 **钳子和爪子**。

这就是 **器（工具）**。

---

## 一句话总结

| 层级 | 含义 | AI里的意思      |
| -- | -- | ----------- |
| 道  | 原理 | 龙虾为什么能做事    |
| 法  | 方法 | 如何设计龙虾做事的方法 |
| 术  | 技巧 | 如何更好地使用龙虾   |
| 器  | 工具 | 使用哪些工具      |

整体逻辑：

**龙虾有能力（道）
→ 给龙虾方法（法）
→ 用技巧沟通（术）
→ 借助工具完成（器）**

---

## 高手成长路径

**善器 → 熟术 → 会法 → 懂道**

先会用工具，再掌握技巧，然后理解方法，最后理解原理。

---

## 本课小结

用「道法术器」的框架理解 AI，可以帮助你：

1. **道（原理）**：理解 AI 为什么能帮你做事（会思考、写代码、用工具）
2. **法（方法）**：学会如何给 AI 安排任务流程
3. **术（技巧）**：掌握与 AI 高效沟通的技巧（清晰指令、拆分任务）
4. **器（工具）**：选择合适的工具组合

从善器开始，逐步成长到懂道，成为 AI 编程高手！
