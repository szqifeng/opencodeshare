# 模型管理

OpenCode 支持同时管理多个 AI 模型，并提供便捷的切换和配置功能。本文将详细介绍模型列表管理、模型切换策略、模型参数配置、性能监控以及模型成本控制。

通过有效的模型管理，您可以根据任务需求、性能要求和预算限制，动态选择最优的模型配置，实现效率和成本的平衡。
