# 安装与配置

详细了解 OpenCode 的安装和配置过程。

## 系统要求

- Node.js 18+
- npm 或 yarn
- Git

## 安装步骤

### macOS / Linux

```bash
npm install -g opencode
```

### Windows

```bash
npm install -g opencode
```

## 配置详解

### 环境变量

| 变量名 | 说明 | 默认值 |
|--------|------|--------|
| OPENCODE_MODEL | 使用的模型 | claude-opus-4-5-thinking |
| OPENCODE_TEMPERATURE | 温度参数 | 0.7 |
