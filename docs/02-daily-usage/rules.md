# 规则设置

OpenCode 的规则系统让您能够精确控制 AI 的行为和输出质量。本文将详细介绍如何配置系统规则、创建自定义规则、管理规则优先级以及优化规则效果。

通过合理设置规则，您可以确保 OpenCode 的输出符合您的品牌调性、技术标准和业务要求，同时避免不当内容的产生。
