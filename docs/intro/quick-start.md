# 快速开始

本指南将帮助你快速上手 OpenCode。

## 安装

```bash
npm install -g opencode
```

## 配置

创建配置文件 `~/.opencode/config.yaml`：

```yaml
model: anthropic/claude-opus-4-5-thinking
temperature: 0.7
```

## 第一个命令

```bash
opencode --help
```
