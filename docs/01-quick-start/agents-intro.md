# Agent

Agent 是 OpenCode 的智能代理系统，每个 Agent 都是一个具有特定角色、规则和能力的 AI 助手，能够自主处理复杂任务并完成工作流编排。

## 什么是 Agent

Agent 是能够自主感知、推理和行动的智能实体，它们：

- 具有明确的角色定位
- 遵循特定的行为规则
- 拥有专属的工具和能力
- 能够自主规划任务步骤
- 可以与其他 Agent 协作

## Agent 的特点

### 1. 自主性

Agent 能够自主做出决策：

```bash
用户：帮我优化项目性能

Agent（代码优化专家）：
✅ 正在分析项目...
✅ 发现 3 个性能瓶颈：
  1. 数据库查询未优化
  2. 图片资源过大
  3. 未使用缓存

💡 优化建议：
  1. 添加索引优化查询
  2. 压缩图片资源
  3. 配置 Redis 缓存

🚀 开始优化...
```

### 2. 规则驱动

Agent 严格遵循预设规则：

```yaml
agent:
  name: "代码审查专家"
  rules:
    - 代码必须符合团队规范
    - 每个函数必须有注释
    - 复杂度不能超过 10
    - 必须处理所有异常
```

### 3. 工具集成

Agent 可以调用各种工具：

```yaml
agent:
  tools:
    - "file.read"      # 读取文件
    - "file.write"     # 写入文件
    - "bash.execute"   # 执行命令
    - "web.search"     # 搜索网络
    - "database.query" # 查询数据库
```

## 可用 Agent

### General（通用代理）

默认的通用智能代理，适用于广泛的任务。

**能力：**
- 自然语言理解
- 任务规划和分解
- 工具选择和调用
- 多步骤任务执行

**适用场景：**
- 日常对话和问答
- 多步骤复杂任务
- 需要灵活适应的任务

---

### Explore（探索代理）

快速探索和理解代码库的专用代理。

**能力：**
- 文件和目录遍历
- 代码模式搜索
- 项目结构分析
- 依赖关系识别

**适用场景：**
- 理解新项目
- 查找代码位置
- 分析架构设计
- 定位问题根源

---

### Code-reviewer（代码审查专家）

严格的代码质量和安全审查专家。

**能力：**
- 代码质量检查
- 安全漏洞检测
- 最佳实践验证
- 性能问题识别

**适用场景：**
- 提交代码前的审查
- 代码质量评估
- 安全性检查
- 性能优化建议

---

### Security-auditor（安全漏洞猎人）

专业的安全审计专家，专注于发现潜在的安全问题。

**能力：**
- SQL 注入检测
- XSS 漏洞扫描
- 认证授权审查
- 敏感数据检查

**适用场景：**
- 安全审计
- 渗透测试
- 合规性检查
- 漏洞修复建议

## Agent 的工作方式

### 任务分解

Agent 会将复杂任务分解为可执行的步骤：

```bash
用户：创建一个用户管理系统

Agent（架构专家）：
📋 任务分解：
  1. 设计数据库模型
  2. 创建 API 端点
  3. 实现用户认证
  4. 添加权限管理
  5. 编写测试用例
  6. 生成 API 文档

✅ 开始执行步骤 1...
```

### 自主决策

Agent 能够根据上下文自主做出决策：

```bash
用户：部署到生产环境

Agent（部署专家）：
🔍 检查项目状态...
✅ 代码已提交
⚠️  测试未通过

💡 决策：暂停部署，先运行测试

🚀 运行测试套件...
✅ 测试通过
✅ 部署到生产环境
```

### 协作执行

多个 Agent 可以协同完成任务：

```bash
用户：完成一个完整的 Web 应用

🤖 [架构师] 设计系统架构...
✅ 架构设计完成

💻 [代码专家] 实现核心功能...
✅ 代码开发完成

🧪 [测试专家] 编写测试用例...
✅ 测试完成

📝 [文档专家] 撰写技术文档...
✅ 文档完成

🎉 所有任务完成！
```

## Agent 的优势

### 1. 专业化

每个 Agent 专精于特定领域，提供更专业的服务。

### 2. 可定制

可以根据需求自定义 Agent 的角色、规则和能力。

### 3. 可扩展

Agent 系统支持无限扩展，可以创建新的专业 Agent。

### 4. 可协作

多个 Agent 可以协同工作，完成复杂的多步骤任务。

## 如何使用 Agent

### 自动选择

OpenCode 会根据任务自动选择合适的 Agent：

```bash
用户：审查这段代码

🤖 自动选择：Code-reviewer Agent

[代码审查专家]
✅ 正在审查代码...
✅ 发现 2 个问题
💡 提供优化建议
```

### 手动指定

可以手动指定使用哪个 Agent：

```bash
用户：@code-reviewer 审查这段代码

🤖 使用指定 Agent：Code-reviewer

[代码审查专家]
✅ 正在审查代码...
```

### Agent 切换

可以在对话中切换 Agent：

```bash
用户：帮我设计系统架构

🤖 [架构师] 正在设计...
✅ 架构设计完成

用户：现在实现代码

🤖 [代码专家] 正在实现...
✅ 代码开发完成
```
