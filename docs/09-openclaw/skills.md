## 🧠 核心技能

### ReAct 模式
ReAct (Reasoning + Acting) 是一种多轮循环推理 + 动作执行的 Agent 架构模式：
- **Thought** - 推理步骤，分析当前状态
- **Action** - 调用工具/API，执行操作
- **Observation** - 获取结果，提供上下文
- **Answer** - 循环终止，输出最终答案
