# 工具配置

OpenCode 提供了丰富的内置工具，并支持用户自定义工具以满足特定需求。本文将详细介绍如何配置和管理 OpenCode 的工具系统，包括内置工具的使用、自定义工具的开发、工具链的搭建以及工具的最佳实践。

通过合理配置工具，您可以大幅扩展 OpenCode 的能力边界，让 AI 助手更好地服务于您的具体工作场景。
