# 命令参考

OpenCode 提供了强大的命令行工具，让您能够通过命令行完成各种操作，包括项目管理、模型配置、任务执行等。本文将详细介绍所有可用的命令、命令参数、使用示例以及命令行最佳实践。

通过掌握命令行工具，您可以实现自动化脚本、CI/CD 集成和批量操作，将 OpenCode 的能力扩展到更广泛的场景。
