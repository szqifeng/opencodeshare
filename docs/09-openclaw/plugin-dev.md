## 🎯 插件类型

OpenClaw 支持 8 种插件类型：
- **Tool Plugin** - 工具插件，提供可复用能力
- **Channel Plugin** - 通道插件，连接外部系统
- **Command Plugin** - 命令插件，封装可执行命令
- **Service Plugin** - 服务插件，提供后台服务
- **Provider Plugin** - 提供者插件，提供外部数据源
- **HTTP Route Plugin** - HTTP 路由插件，接收外部请求
- **Hook Plugin** - 钩子插件，监听系统事件
- **TypedHook Plugin** - 类型化钩子插件
